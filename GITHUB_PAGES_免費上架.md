# GitHub Pages 免費上架

這一版不需要 Render，也不需要信用卡。掃描直接在開啟網頁者的瀏覽器中執行，使用 BingX 公開行情。

1. 在 GitHub 的既有專案頁面，按 `Add file`，選 `Upload files`。
2. 將本資料夾內的 `index.html` 及 `.nojekyll` 拖進去，確定它們與 `main.py` 在同一層，按 `Commit changes`。
3. 在專案上方按 `Settings`，左側選 `Pages`。
4. 在 `Build and deployment` 的 `Source` 選 `Deploy from a branch`。
5. `Branch` 選 `main`，資料夾選 `/(root)`，按 `Save`。
6. 等約一分鐘，重新整理此頁。GitHub 會顯示你的免費網站網址。

提醒：資料夾名稱不要包住檔案；`index.html` 必須在 GitHub 專案最外層。
